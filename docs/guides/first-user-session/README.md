# First User Session Kit

Ziel: 3 bis 5 Nutzer testen AFBM ohne Erklaerung. Die Session prueft, ob neue Nutzer den ersten Spielzyklus verstehen: Dashboard lesen, naechste Aktion finden, Team pruefen, Spiel vorbereiten, Spiel abschliessen, Report verstehen und eine zweite Woche starten wollen.

## Unterlagen

- `testleitfaden.md`: Ablauf fuer Moderator oder Beobachter.
- `nutzer-aufgabenliste.md`: Aufgabenblatt fuer die Nutzer.
- `beobachtungsbogen.md`: Live-Notizen waehrend der Nutzung.
- `feedbackformular.md`: Fragen nach der Session.
- `auswertungsvorlage.md`: Zusammenfassung ueber alle Sessions.
- `erfolgskriterien.md`: Klare Kriterien fuer Gruen/Gelb/Rot.

## Empfohlenes Setup

- 3 bis 5 Personen.
- 25 bis 35 Minuten pro Person.
- Start auf dem Savegame-Dashboard.
- Keine Vorab-Erklaerung der UI, Begriffe oder optimalen Entscheidungen.
- Hilfe nur geben, wenn der Nutzer komplett blockiert ist. Jede Hilfe als Beobachtung notieren.

## Rollen

Moderator:
- liest Begruessung und Aufgaben vor
- beantwortet keine inhaltlichen Fragen zum Spielsystem
- achtet auf Zeit und Reihenfolge

Beobachter:
- notiert Suchbewegungen, Missverstaendnisse, Abbrueche und starke Momente
- dokumentiert jede Hilfestellung

Nutzer:
- denkt laut
- versucht alle Aufgaben ohne Erklaerung zu loesen
- fuellt danach das Feedbackformular aus
