Post Game Motivation Report

Aufgabe: Nach dem Match Report soll sofort ein klarer Verbesserungsimpuls entstehen. Der Nutzer soll nicht fragen: Was jetzt? Sondern denken: Das fixe ich als Naechstes.

Executive Summary

Direkt unter dem Score erscheint jetzt ein einzelnes Ziel fuer die naechste Woche. Das Ziel wird aus dem groessten sichtbaren Problem im Report abgeleitet und fuehrt direkt zur passenden Aktion: Depth Chart oder Roster.

Zielregeln

Signal Ziel Aktion
2+ Turnovers Turnovers reduzieren Depth Chart pruefen
Deutlich weniger Passing Yards als der Gegner Passing stabilisieren QB und Receiver pruefen
Gegner mit mindestens 40 Yards Vorsprung Defense verbessern Roster ansehen
Red-Zone-Trips ohne Touchdown Red Zone schaerfen Red-Zone-Rollen pruefen
Kein dominanter Bruch Staerke bestaetigen Aufstellung bestaetigen
