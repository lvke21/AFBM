UI Language Polish Report

Aufgabe: Technische Begriffe im Spieltag-, Aufstellungs- und Report-Flow durch kurze, spielerische Sprache ersetzen. Ziel ist, dass Nutzer die Situation wie eine GM-Entscheidung lesen, nicht wie einen Systemzustand.

Executive Summary

Die sichtbarsten technischen Labels wurden im Game Preview, Depth Chart, Match Start, Live Screen, Match Report, Inbox und Dashboard geglaettet. Interne Kennzeichnungen wie DEV_AUTO_FILL bleiben intern erhalten; die UI spricht nun von Ersatzspielern, Aufstellungsrisiko und Spieltag-Bereitschaft.

Vorher Nachher Wirkung

Lineup unvollstaendig - Performance kann leiden Dein Team ist nicht optimal aufgestellt. Ersatzspieler muessen einspringen. Kurz, klar und als Football-Risiko lesbar.
