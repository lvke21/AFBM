## Verwendung

- Abschluss eines APs dokumentieren.
- Tests, Status und bekannte Einschraenkungen festhalten.
- Folge-APs nur referenzieren, wenn sie inhaltlich aus dem Report ableitbar sind.

## Naming

Neue Dateien sollen dem Muster `phase-<area>-ap<nr>-report.md` folgen, zum Beispiel `phase-roster-management-ap43-report.md`.
