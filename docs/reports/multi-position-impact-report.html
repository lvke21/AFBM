Multi Position Impact

Aufgabe: Lineup-Entscheidungen und Positionsgruppen sollen nicht mehr nur ueber QB/Passing sichtbar werden. Der Spielbericht und die Live Simulation sollen mindestens drei Positionsgruppen spuerbar rueckmelden.

Executive Summary

Der Post-Game Report zeigt jetzt drei zusaetzliche Positionsgruppen als eigene Team-Impact-Karten: Run Game, WR Matchup und Defense Pressure. Live und Key Moments erhalten zudem gelegentliche Positionsimpact-Hinweise aus vorhandenen Drive-Daten, zum Beispiel RB/Run Game bei starken Rush-Drives, WR Matchup bei Passing-Drives und Defense Pressure bei Stops oder Turnovers.

Umsetzung

RB / Run Game Report-Werte: Run Game stark, Run Game stabil oder Run Game schwach. Basis sind Rushing Yards, Rush Edge und Starter-OVR.
WR / Passing Matchup Report-Werte: WR Matchup Vorteil, neutral oder schwach. Basis sind Passing Yards, Big Plays, Passing Edge und WR/TE-Starter.
DL / Defense Report-Werte: Defense Pressure stark, stabil oder schwach. Basis sind Drive Stops, gegnerische Turnover und Front-Seven-Starter.
