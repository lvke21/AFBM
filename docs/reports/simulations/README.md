# Simulation Reports

## Zweck

Dieser Ordner enthaelt versionierte Simulationsberichte, die als Referenz oder Abschlussnachweis erhalten bleiben sollen.

## Dateien

- [balancing-round-ap24-ap32-final-report.html](./balancing-round-ap24-ap32-final-report.html) - AP-/Phasenabschluss und Statusnachweis
- [extended-season-balance-results.html](./extended-season-balance-results.html) - Simulations- oder Balancing-Snapshot
- [extended-season-balance-results.json](./extended-season-balance-results.json) - Simulations- oder Balancing-Snapshot
- [play-simulation-stability-2026-04-22.md](./play-simulation-stability-2026-04-22.md) - Simulations- oder Balancing-Snapshot
- [validation-simulation-final-report-2026-04-22.md](./validation-simulation-final-report-2026-04-22.md) - Validierungsnachweis

## Abgrenzung

Neue, wiederholbar generierte Outputs gehoeren nach `reports-output/simulations/`. Nur kuratierte Snapshots werden hier abgelegt.

## Naming

Neue Simulationsberichte verwenden `sim-<scenario>-<runs>-report.md`, zum Beispiel `sim-8teams-20games-report.md`.
