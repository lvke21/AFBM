# Multiplayer Fantasy Draft Player Pool

## Summary

The initial Multiplayer Fantasy Draft now uses a deterministic fictional player
pool generated by `createFantasyDraftPlayerPool()` in
`src/lib/online/online-league-service.ts`.

Seed: `afbm-online-fantasy-draft-v1`

For a 16-team league, each team drafts a 41-player MVP roster. The generated pool
contains 792 players, which covers the 656 required drafted roster spots plus
roughly 20% reserve by position.

## Generation Model

- IDs are deterministic and derived from the fixed seed, league id, position and
  position index.
- Names are fictional and assembled from internal first-name and last-name lists
  plus a position label.
- Ages are generated in a controlled range:
  - regular positions: 21-34
  - K/P: 23-36
- Ratings are generated from deterministic tiers:
  - elite: 88-94
  - starter: 80-87
  - rotation: 68-79
  - depth: 58-67
  - K/P use a narrower specialist distribution from 62-90
- Each player receives the required `OnlineContractPlayer` fields:
  `playerId`, `playerName`, `position`, `age`, `overall`, `potential`,
  `developmentPath`, `developmentProgress`, `xFactors`, `contract`, `status`.
