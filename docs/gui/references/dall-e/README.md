# Dall-E GUI References

## Zweck
Diese Bilder dienen ausschließlich als visuelle Referenz für das GUI-Design.

Sie werden NICHT direkt in der Anwendung verwendet.

## Verwendung
- Ableitung von Layouts
- Definition von Komponenten
- Festlegung von Farbwelt und Typografie
- Inspiration für UX-Strukturen

## Stilrichtung
- Pro Analyst Sports UI
- Dark Theme (Navy / Charcoal)
- High Readability
- Data-driven Layouts
- Modern Sports Game Accents (subtle, nicht verspielt)

## Wichtige Regeln
- Keine 1:1 Umsetzung der Bilder
- Keine direkten Image-Imports in die App
- UI wird als Komponenten-System gebaut
