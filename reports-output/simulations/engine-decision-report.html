Neue Engine Entscheidungen im Game Report

Dieser Bericht erklaert die neuen Engine-Entscheidungen fuer Anwender und Entwickler. Der normale Game Report soll kurze Zusammenfassungen zeigen; der Developer Debug Mode darf die Detailtabellen anzeigen.

Was sichtbar wird

BereichAnzeige im ReportErklaerung fuer Anwender
Play CallOffense-Call und Defense-CallZeigt, welche Idee beide Seiten vor dem Play hatten.

QB DecisionDecision Time, verfuegbare Targets, gewaehltes TargetZeigt, warum nicht jede Route anspielbar war.
CoveragePress Result, Separation ResultErklaert, ob Receiver oder Defender das Matchup gewonnen haben.
Run GameRun Lane QualityZeigt, ob die Line eine Lane erzeugte oder der RB selbst kreieren musste.

Bewertung

Die Anzeige bleibt bewusst zweistufig: Anwender sehen kurze Football-Erklaerungen, Entwickler erhalten im Debug Mode die Faktoren und Werte. Dadurch wird der Game Report nachvollziehbarer, ohne die GUI mit Rohdaten zu ueberladen.
