Simulation API Documentation

Standalone API fuer spaetere Frontend-Integration. Die Endpoints verwenden die bestehende Match-Engine und erzeugen echte Simulationsergebnisse.

Status: GRUEN

Endpoint-Uebersicht

FunktionHTTPPfadZweck
createGamePOST/api/simulation/createGameErzeugt ein Spiel mit Teams, Seed, Gameplan und Settings.
simulateGamePOST/api/simulation/simulateGameSimuliert ein zuvor erzeugtes Spiel.
getGameResultGET/api/simulation/getGameResult?gameId=...Liefert Score, Winner, Team Stats, Player Stats und Drive Summary.
getDriveLogGET/api/simulation/getDriveLog?gameId=...Liefert Drive-by-Drive-Informationen.
getStatsGET/api/simulation/getStats?gameId=...Liefert frontend-taugliche Team-/Player-Stats, Leaders und Summary.

Fehlerfaelle

StatusCodeBedeutung
400INVALID_INPUT, INVALID_JSON, MISSING_GAME_IDRequest ist ungueltig oder unvollstaendig.
404GAME_NOT_FOUNDDie angefragte gameId ist unbekannt.
409GAME_NOT_SIMULATEDResult, DriveLog oder Stats wurden vor der Simulation abgefragt.
409GAME_ALREADY_EXISTSEine explizit gesetzte gameId existiert bereits.
